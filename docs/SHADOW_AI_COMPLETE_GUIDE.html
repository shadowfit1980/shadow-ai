11. Development & Extension Guide

11.1 Adding a New Feature

  1. Create the backend service in src/main/ai/[category]/
  2. Add IPC handlers in src/main/ipc/advancedHandlers.ts
  3. Expose API in src/main/preload.ts
  4. Create UI component in src/renderer/components/[category]/
  5. Add tests in tests/

11.2 Key Files to Understand

File Purpose
src/main/index.ts Main process entry point
src/main/preload.ts IPC bridge and API exposure
src/main/ipc/advancedHandlers.ts Queen 3 Max IPC handlers
src/renderer/App.tsx Main React application
src/renderer/utils/codingIntent.ts Slash command parsing

11.3 Coding Standards

  • TypeScript strict mode enabled
  • React functional components with hooks
  • Singleton pattern for backend services
  • IPC channel naming: category:action
  • Comprehensive error handling with typed errors
