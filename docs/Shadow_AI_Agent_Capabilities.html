Executive Summary

Shadow AI is an advanced AI-powered coding assistant built on Electron with a multi-model architecture. It combines 15+ specialized agents, 50+ services, and deep IDE integration to provide autonomous development capabilities across web, mobile, game, and desktop platforms.

Quick Start for Development

Adding a New Agent

  1. Create in src/main/ai/agents/specialist/
  2. Extend SpecialistAgent base class
  3. Define capabilities array
  4. Implement execute(task) method
  5. Export in index.ts
  6. Add IPC handler in domainAgentHandlers.ts
  7. Expose in preload.ts and shadowAPI.ts

Adding a New Service

  1. Create in src/main/services/
  2. Implement as singleton class
  3. Add IPC handlers for methods
  4. Create React hook in src/renderer/hooks/

Adding UI Components

  1. Create .tsx and .css files
  2. Use existing hooks for API access
  3. Follow dark theme design patterns
  4. Export from component index
